<img> 태그배우기

브라우져에 이미지를 삽입하는태그
<br>처럼 </img> 로 닫지않는태그이다.
<img>태그의 가장 기본적인 속성은 이미지 경로를 지정하는 src 속성입니다.
src 속성에 이미지 경로를 입력하고 width(가로), height(세로) 속성으로 크기를 지정한다. 이때 이미지 파일이 src 속성에서 지정한 경로에 없을시, 이미지는 출력되지 않거나 엑스박스가 뜬다
