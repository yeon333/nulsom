Iframe Element


Iframe의 의미는 The iframe element represents a nested browsing context. 라고 메뉴얼에 나와있습니다.
알기쉽게 말하자면 자신의 웹 사이트에 다른 사이트에 있는 내용을 가져와 보여 주는 것 입니다.






iframe의 단점


iframe에는 심각한 취약점이 하나 있습니다. 그것은 바로 사용자를 iframe으로 사이트를 통째로 유도하여 해커가 만든 악성코드 배포지로 유도하는 것인데요.
이로인해 많은 피해가 발생했습니다.
