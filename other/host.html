원레 인터넷 주소창은 특정 ip 로 되어있습니다 하지만 인간들이 보기 편한 영어로 바꿔두면 자동으로 ip로 접속됩니다 하지만 컴퓨터는 모든 웹사이트에 ip를 알수없습니다. 그래서 나온것이 DNS(Domain name system)입니다. 이 것은 인간이 영어로 된 주소창을 쓰면 ip를 알려주는 기능인데 인터넷이 처음 나왔을 당시에는 웹사이트가 얼마 없어서 컴퓨터에 모든 웹사이트 주소 ip를 썼는데 그 파일이 바로 host 파일입니다.
현재 윈도우 OS 기준으로 host 파일은 C:\\Window\System32\drivers\etc 경로에 있습니다
현재 dns 서버덕분에 이 host 파일은 거의 안쓰이는데 크래킹용으로 사용되곤 합니다.
그럼 관련 실습을 해보겠습니다.

먼저 국내 포털사이트 네이버의 ip 주소를 알아봅니다.

host 파일에 네이버 ip를 쓰고 구성고등학교의 주소를 적습니다.

저장후 인터넷 주소창에 구성고등학교 주소를 아무리 써도 네이버로 이동됩니다.
실제로 크래커들은 네이버 ip가 아닌 자신이 만든 네이버 변조사이트로 이동시켜서 크래킹을 하기도 합니다
